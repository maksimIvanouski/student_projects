# Student Projects

This repository contains my student projects and programming assignments completed during university studies and additional courses.

The projects are organized by semester and topic.

The repository includes small practice tasks and console projects written mainly in Python and C++.


## Topics

- Python basics
- Data types
- Lists, tuples, dictionaries, and strings
- Functions and decorators
- Recursion
- Object-oriented programming
- Exceptions
- C++ console programs
- Simple game and shop simulations

## How to Run

For Python files:

```bash
python file_name.py
```

For C++ files:

```bash
g++ file_name.cpp -o program
./program
```
